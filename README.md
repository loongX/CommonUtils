### 基本功能记录

#### util相关
- App基本信息工具 `AppInfoUtil`
- 状态栏相关（透明状态栏、状态栏着色） `StatusBarUtil`
- 日志打印工具 `LogUtil`
- 软键盘显示隐藏工具 `KeyboardUtil`
- 文件管理工具 `FileUtil`
- 屏幕尺寸及相关UI工具 `UIUtil`
- 单位转换及基本App工具 `AppUtil`
- 不同手机功能相关工具 `RomUtil`
- 矩阵计算工具 `MatrixUtil` 
- 图片裁剪或保存工具 `ImageUtil`

#### UI相关
- 流式布局 `FlowLayout`
- 轮询banner `WheelViewPager`
- 图片裁剪 `EditImageView`
